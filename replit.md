# The Guard - Personal Safety AI

## Project Overview
A documentation repository for "The Guard" — a technical development roadmap for an AI-powered personal safety application.

## Files
- `cover_guard.html` — HTML/CSS cover page (served as homepage)
- `generate_guard_body.py` — ReportLab script to regenerate the body PDF
- `The_Guard_Technical_Development_Roadmap.pdf` — The complete 15-page roadmap
- `serve.py` — Python HTTP server to serve the cover page

## Architecture
- **Server:** Python 3.12, http.server on port 5000, host 0.0.0.0
